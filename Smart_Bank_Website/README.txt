
Smart Bank - Demo Website
-------------------------
Files:
- index.html (Home)
- services.html (Services overview)
- contact.html (Contact form - demo)
- login.html (Demo login page)
- assets/styles.css
- assets/scripts.js

How to use:
1. Unzip the package.
2. Upload the folder to any static hosting (GitHub Pages, Netlify, Vercel) or open index.html locally for testing.
3. The site is a demo and contains no backend - forms are client-side only.
